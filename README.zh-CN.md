# color-lighten-tool
 
一个让颜色变淡的工具，类似于：输入色值 + 透明度 + 白色背景 = 输出色值。

Language : [English]((./README.md)) | 中文



📦 格式多（UMD module + ESModule + Typescript）.



### 安装

```
npm i color-lighten-tool
```



### API

`function colorLightenTool(color: string, opacity: number): string`
